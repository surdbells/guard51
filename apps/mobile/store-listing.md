# Guard51 — Store Listing

## App Name
Guard51 — Security Guard App

## Short Description
Clock in/out, patrol tours, panic alerts, and incident reporting for security guards.

## Full Description
Guard51 is the companion mobile app for security guards managed through the Guard51 platform.

Features:
• Clock In/Out with GPS verification
• Panic button with instant dispatch notification
• Tour checkpoint scanning (QR codes)
• Incident reporting with photo evidence
• Passdown log viewing and acknowledgment
• Real-time chat with supervisors and dispatch
• Offline mode — actions sync when connection restores

Built for security companies in Nigeria using the Guard51 SaaS platform.
